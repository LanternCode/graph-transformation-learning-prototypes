# Graph Transformation Learning Prototypes

Standalone research prototypes and directional baseline experiments developed during my PhD work on graph transformation learning, graph machine learning, and benchmark-driven model evaluation.

These scripts predate the current organised research framework, which is currently private while associated work is under review. They were not used as official paper reproducibility artifacts, but they informed the development of later benchmark and evaluation pipelines.

The repository is public because these experiments are independent of the under-review paper code and may later be refactored as extensions to the full framework.

## Status

Research prototype / pre-framework baseline code. Public, but not actively maintained as a standalone package.

## Relationship to my PhD work

These experiments provided directional baselines for graph transformation learning research. They helped test whether later framework development was empirically sensible, but they are separate from the main paper reproducibility repository.

## What this demonstrates

- graph/ML experimentation
- baseline model implementation
- hyperparameter tuning
- task-specific evaluation
- early extension ideas for the full framework

## What this is not

This is not the final research framework and not the official reproducibility package for any paper currently under review.
